# Sber landing

Статический лендинг на два экрана для эфира Сбера.

## Структура

- `index.html` — разметка лендинга и модальной формы регистрации.
- `src/styles.css` — вся визуальная система и адаптив.
- `src/main.js` — открытие формы и фронтовое подтверждение заявки.
- `public/assets/` — рабочие ассеты: логотип, фото эксперта и сгенерированный горный фон.
- `public/source/` — исходные файлы, которые были переданы в проект.

## Запуск

```bash
python3 -m http.server 4173
```

После запуска страница доступна на `http://127.0.0.1:4173/`.
